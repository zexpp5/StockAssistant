## 一、当前已知的免费数据卡点

### 1.1 候选发现 / [discover_candidates.py](../discover_candidates.py)

| 现象 | 影响 | 根因 | 临时解 |
|---|---|---|---|
| 中国 A 股 `t.info` 返回 HTTP 404 | 市值过滤对 A 股不生效，所有 A 股都通过 cap filter | yfinance 对 .SS / .SZ 的 info endpoint 不全 | 现状先全部放过；对 CN 不打 cap filter |
| 非美股市值用本币显示 | 看板"市值 ($B)"列对 .TW / .HK / .SZ 是误导（如 2308.TW 显示 $5715B 实际 ≈ $180B USD） | yfinance marketCap 字段是本币，未做汇率换算 | UI 暂未修；标注用户手动换算 |
| 非美股分析师评级几乎全 0 | 综合 z-score 里"分析师上修"维度对 CN 全部为 0，影响排序 | yfinance 对非美股 analyst 数据基本不收 | 让 z-score 退化为 3 因子（F + 动量 + PEAD） |
| 部分 A 股（科创 688xxx / 创业 300xxx）`t.financials` 拿不全 | Piotroski F-Score 算不出（None）→ 这只股票被合成阶段 fillna(0) 拉平 | yfinance 对小盘 A 股财报覆盖差 | 标注 F=None 直接跳过该股 |

### 1.2 daily_picks_v5（学术因子选股）

[daily_picks_v5.py:46](../daily_picks_v5.py#L46) 写死了：
```python
def is_us_ticker(code):
    return code.isalpha() and 1 <= len(code) <= 5
```

**watchlist 78 只里 ~30 只非美股全部跳过打分**。这意味着用户飞书 watchlist 里手编的中际旭创 / 北方稀土 / 阿里 / 腾讯 等都没进入 v6 排序池。**v6 方案 A 仅对美股有效，CN/HK 是手编固定权重**。

### 1.3 估值字段（Forward PE / PEG / 利润增速）

`enrich_watchlist` 流程靠 yfinance + Finnhub 拉前瞻 PE / PEG，
对 A 股很多直接返回 `None`。看板"估值视角"散点图非美股股票全部缺席。

### 1.4 13F 机构持仓只能看美股

[track_13f.py](../track_13f.py) 用 SEC EDGAR，**结构上就只覆盖美股**。HK / A 股没有等价的"机构持仓 13F"披露 → 高瓴 / 段永平在 A 股的持仓追不到。

### 1.5 财务深度研究（fundamental_deep）

[stock_research/core/fundamental_deep.py](../stock_research/core/fundamental_deep.py) 用 FMP 取 5-10 年三表做深度因子（杜邦/M-Score/Z-Score/8 项盈利质量）。FMP 免费档对单股 NVDA 跑通；批量同业对标时部分 endpoint 限流（key-metrics-ttm 偶发 402）。

### 1.6 B 路线个股深度研究系统（Phase 1-4，2026-05-10 落地）

详见 [docs/2026-05-10_B路线个股深度研究系统.md](./2026-05-10_B路线个股深度研究系统.md)。当前卡点：

| 模块 | 卡在哪 | 解锁条件 |

| Phase 1B 同业排名置信度 [peer_compare.py](../stock_research/core/peer_compare.py) | FMP 免费档同业候选少（NVDA 实测只剩 1 家），key-metrics-ttm 偶发 402 | FMP Starter $35/月（同业候选广 + endpoint 全） |
