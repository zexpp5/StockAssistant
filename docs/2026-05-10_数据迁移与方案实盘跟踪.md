重新设计为 **forward-only**：

| 字段 | 含义 |
|---|---|
| `inception_date` | DuckDB `snapshots/optimize/plan_v6` 最早写入日期 = v6 方向定下来那天 |
| `baseline_date` | history 中 ≤ inception 的最后一个交易日，NAV=100 锚点 |

| tracked window | 锁定后所有交易日（紫色实线 + 锁定竖线标记） |

新逻辑只在 tracked 段算指标。今天首日 tracked = 0 天（plan 锁定在周六，市场没开），等 5-11 周一收盘后第一个数据点出现。
