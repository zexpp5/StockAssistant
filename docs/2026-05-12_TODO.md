  - 美股老线 daily_picks_v5 ✅ 接了 yfinance `upgrades_downgrades`，[score_analyst()](scripts/lib/early_signals.py#L192-L207) 0-15 分加成
  - A 股 / 港股 picks ❌ 没接（数据源没有等价物）
  - Finnhub `recommendation_trends`（强买/买/持有/卖快照）❌ 拉了但只在 watchlist 展示，**没进打分**
  - Finnhub `price_target`（目标价共识）❌ 同上，只展示
- **学术判断**：
  - "评级动作"（上调/下调）有 alpha — Womack 1996、Barber 2001
  - "评级快照"（强买总数）alpha 在 2010 后衰减 — 全员强买的反而平庸

- **建议补法**（非必做，等回头看）：
  - 美股 picks 加一个 0-10 分加成：`(target_median / current - 1) > +30% → 10 分`
  - **不要**接 `recommendation_trends`（容易被分析师群体追涨杀跌带偏）
  - A 股 / 港股保持现状（数据贵 + 北向 / 龙虎榜 alpha 已经更强）
- **工作量**：半天（接 Finnhub `price_target` → 美股 picks composite）
- **风险**：免费 Finnhub 层 `price_target` 经常 403，需要付费层稳定覆盖；先评估命中率再决定
