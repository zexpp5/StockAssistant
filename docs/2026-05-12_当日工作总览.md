### 自检清单（每次新接入约束器 / fallback 路径必跑）

- [ ] 调用方有显式 `if enable_X` 或权重 > 0 守卫，weight=0 时**真不跑**
- [ ] fallback 路径返回与正常路径**同类型**（dataclass vs dict 不能混）
- [ ] smoke test 跑一次 `enable=False` 路径，验证所有下游属性访问 OK
- [ ] commit message 标注守卫位置 + fallback 类型对齐
