# 周末复盘规则

更新时间：2026-05-25

## 定位

`weekly_self_review.py` 只做 **「模型推了什么」vs「你实际做了什么」** 对齐，不做因子归因，不自动改持仓或 watchlist。

## 数据来源（PIT）

| 侧 | 表/文件 | 规则 |
| --- | --- | --- |
| 模型推 | `recommendation_runs` + `recommendation_picks` | 按 `run_date` 周内每日 run；Top N 默认 10；只用 `universe_scope=system_tech_universe` |
| 你做 | `state_backup/state_*.json` → `real_holdings` | 周初 vs 周末 shares diff；无备份时用当前 `real_holdings` |
| 体检 | `real_holding_review_items` | 周内最近 `action_label` |

禁止用「今日最新 picks」回填历史周。

## 四类对齐

1. **错过**：周内曾进模型 Top N，周末未持仓且周内无增持。
2. **没听话**：周内增持，且体检 verdict ∈ {风险复查, 减仓观察}。
3. **对齐**：持仓/增持与模型方向大体一致。
4. **逆势赚钱**（子集）：没听话且 `pick_outcomes` 5d > 0（仅展示，不作策略结论）。

## 产物

- `data/latest/weekly_self_review_YYYY-Www.json`
- `data/latest/weekly_self_review_latest.json`
- `docs/letters/weekly_self_review_YYYY-Www.md`
- 飞书（可选）：`FEISHU_WEEKLY_WEBHOOK` 或 `FEISHU_BRIEF_WEBHOOK`

## 调度

```bash
python3 -m stock_research.jobs.weekly_self_review --push
python3 -m stock_research.jobs.weekly_self_review --dry-run   # 只写文件
```

`daily_refresh.sh` 在周日（`DOW=7`）自动执行并 `--push`。

## 今日决策台展示

`build_stock_dashboard_html.py` 读取 `data/latest/weekly_self_review_latest.json`，在今日决策台卡片下方展示「上周复盘」摘要。
